# Maternal Health App

A comprehensive Flutter application designed for maternal health monitoring in rural areas.

## Features
- **Patient Module**: Phone/OTP registration, diet plans, vaccination tracking, and medicine schedules.
- **ASHA Worker Module**: Patient monitoring, health alerts, and visit management.
- **Bilingual Support**: English and Hindi support for enhanced accessibility.
- **Premium UI**: Designed with maternal-specific branding and high-accessibility visual cues.
